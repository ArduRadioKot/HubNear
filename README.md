# DeVIZ

Frontend-часть проекта **dViz / HubNear** на **React + TypeScript + Vite**.

## Описание проекта

**HubNear** — мобильный сервис для быстрых сборов на активности рядом.
Основной фокус — спорт и групповые активности, где важно набрать людей к конкретному времени.

Что делает пользователь:
- открывает приложение;
- видит сборы рядом;
- вступает в существующий сбор или создает свой.

Пример сценария:

> «Волейбол сегодня в 19:00, парк Горького, нужно 12 человек, уже 8, уровень любительский, набор до 17:30».

Логика подтверждения:
- если минимум участников набран до дедлайна, сбор получает статус **подтвержден**;
- если минимум не набран до дедлайна, сбор не подтверждается.

## Стек

- React 18
- TypeScript
- Vite 6
- Tailwind CSS 4 (`@tailwindcss/vite`)
- UI-компоненты на базе Radix UI

## Быстрый старт

```bash
cd HubNear
npm install
npm run dev
```

После запуска приложение будет доступно по адресу, который покажет Vite (обычно `http://localhost:5173`).

## Скрипты

В каталоге `HubNear/`:

- `npm run dev` — запуск dev-сервера
- `npm run build` — production-сборка
- `npm run preview` — локальный просмотр production-сборки

## Структура

```text
HubNear/
  src/
    app/
      App.tsx                # основной экран/роутинг по состояниям
      components/ui/         # UI-компоненты
    styles/                  # глобальные стили, тема, шрифты
    main.tsx                 # точка входа
  public/
  vite.config.ts             # конфиг Vite + alias @ -> src
```

## Дополнительно

В репозитории есть папка `firstfront/` с HTML-черновиками и референсами для верстки.
Если нужна чистая HTML-верстка отдельных экранов, можно брать за основу материалы из `firstfront/ref/`.
